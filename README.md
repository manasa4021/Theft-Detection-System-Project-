# Theft Detection System

## Overview
A computer vision based theft detection system that identifies suspicious behavior from surveillance videos.

## Features
- Person detection
- Suspicious activity detection
- Video processing
- Alert generation
- Web interface for monitoring

## Technologies Used
- Python
- OpenCV
- Flask
- NumPy

## Project Structure
- src/ : Core detection modules
- templates/ : Web pages
- tests/ : Testing scripts
- docs/ : Documentation

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

python app_web.py
